# Lion and Lamb AI website

Open `index.html` for a quick local preview. For the most accurate preview, run a local static server from this folder.

## Before launch

Use `PROJECT_CHECKLIST.md` as the source of truth. Replace all clearly marked placeholders, update `example.com`, connect forms and booking, and review the legal drafts.

## Integration safety

`integrations.config.example.js` lists the prepared connection points. Only public identifiers belong in browser files. OpenAI, Claude, private n8n webhooks and all secret keys must be kept on a server-side service.
